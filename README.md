## 기본개념

- 하루동안 매 시간 얼마나 깨어있는지를 스스로 시간 체크하는 앱을 만든다.
- 에너지감각을 유지한 시간이 1시간마다 얼마나 되는가. 이것을 매시간별로 기록한다. 그리고 하루. 매주. 통계를 낸다.
- flutter로 개발하고 firebase db를 사용.
- android와 ios app publish 한다

## 유지 앱 개발 기획 및 UI 설계

유지 앱 개발 기획안과 UI 제안을 준비해 드립니다. 사용자가 매 시간 에너지를 느꼈던 시간을 기록하는 앱입니다.

**유지 앱 개발 기획안 및 UI 제안**

**1. 앱 개요 및 목표**

- **앱 이름:** Joongshim uji app (중심 유지 App)
- **플랫폼:** iOS / Android (Flutter 개발)
- **데이터베이스:** Firebase (예: Firestore)
- **카테고리:** 라이프스타일
- **목표:**
- 하루 동안 에너지 감각을 느끼며 깨어 있었던 시간을 간편하게 기록하고 시각화합니다.
- 일일 에너지 활용률(퍼센트)을 계산하여 하루를 얼마나 활기차게 보냈는지 직관적으로 파악할 수 있도록 돕습니다.
- 
- 사용자가 자신의 에너지 흐름 패턴을 이해하고, 생활 루틴을 개선하며 번아웃을 예방할 수 있도록 지원합니다.
